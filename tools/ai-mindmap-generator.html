URL Parameters

Use URL parameters to programmatically generate mind maps. Perfect for integrations and embedding.

text string

The topic, prompt, or keyword to generate a mind map from.

mindmap boolean

Set to true to enable mind map generation mode.

name string (optional)

Custom name for the generated mind map graph.

Example URL

https://app.thinkmachine.com/new?text=Artificial+Intelligence&mindmap=true&name=AI+Overview
