How It Works

Markdown to Mindmap in 5 steps

01

Paste or upload

Paste your Markdown content directly into the text area, or upload a .md file using the file upload zone.

02

Structure parsed

Our parser analyzes your Markdown structure — identifying headings, lists, paragraphs, and nested content.

03

Headings become branches

Each heading level maps to a branch depth in your mind map, preserving the original document hierarchy.

04

Content visualized

Lists, paragraphs, and inline content are attached as leaf nodes, giving you a complete visual representation.

05

Navigate your map

Explore your interactive mind map — click nodes to expand, search for concepts, and discover how your content connects.

URL Parameters

Use URL parameters to programmatically generate mind maps. Perfect for integrations and embedding.

text string

The Markdown content to convert into a mind map.

mindmap boolean

Set to true to enable mind map generation mode.

name string (optional)

Custom name for the generated mind map graph.

Example URL

https://app.thinkmachine.com/new?text=%23+My+Document%0A%23%23+Section+One&mindmap=true&name=Markdown+Map
