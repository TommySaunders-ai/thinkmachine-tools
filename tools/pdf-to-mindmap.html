Documentation

URL Parameters

Use URL parameters to programmatically generate mind maps. Perfect for integrations and embedding.

text string

The extracted text content from a PDF to convert into a mind map.

mindmap boolean

Set to true to enable mind map generation mode.

name string (optional)

Custom name for the generated mind map graph.

Example URL

https://app.thinkmachine.com/new?text=Extracted+PDF+Content&mindmap=true&name=PDF+Map
